## Веб-клиент

Добавлен веб-интерфейс, повторяющий функциональность Telegram-бота и умеющий отображать CSV-таблицы и изображения прямо в чате. Подробности по запуску: [frontend/README.md](frontend/README.md).

Запуск:
```
python -m http.server --directory frontend 4173
```
